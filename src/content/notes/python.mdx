# Python On Windows
## 原生 Python 安装
Windows 的 Python 安装可以直接使用微软应用商店：
- [Python 的安装](https://apps.microsoft.com/detail/9nj46sx7x90p?activetab=pivot%3Aoverviewtab&rtc=1&hl=zh-cn&gl=CN)
- [Python 的使用](https://learn.microsoft.com/zh-cn/windows/python/beginners)

如果无法访问微软应用商店，你可能需要使用一些科学上网的工具。

## Conda 的安装
在 [官网](https://repo.anaconda.com/miniconda/Miniconda3-latest-Windows-x86_64.exe) 下载对应的安装包，然后执行安装包即可。

# Python On Linux（以 Ubuntu 为例）
## 原生 Python 安装
Ubuntu 22.04 应该已经安装了 Python 3，可以通过以下命令查看版本：
```bash
python3 --version
```
即可看到。

值得注意的是，直接输入 `python` 可能会找不到 Python 3，这是因为没有设置软链接的原因，这时候你需要输入 `python3`。

当然，我们经常会使用到虚拟环境，这时候使用 Conda 会更合适。

## Conda 的安装
我们不推荐使用 Anaconda，而是推荐使用 Miniconda，因为 Miniconda 更加轻量级，而且可以自定义安装。

在 [官网](https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh) 下载对应的安装包，然后执行以下命令：
```bash
bash Miniconda3-latest-Linux-x86_64.sh
```

然后一路回车，只有两点可能需要注意：
1. 选择安装路径。默认安装在 `~/miniconda3`，你可以选择其他路径。
2. 是否将 Conda 加入 PATH。如果你选择了不加入，那么你需要手动将 Conda 加入 PATH。建议选择加入。

安装完成后，你需要重启终端，或者使用 `source ~/.bashrc` ，然后执行以下命令：
```bash
conda --version
```
如果看到版本号，那么说明安装成功。
